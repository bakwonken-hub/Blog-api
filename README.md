# Blog API

API RESTful pour la gestion d'articles de blog.

## 📋 Prérequis
- Node.js (v14+)
- npm ou yarn
- MySQL

## 🚀 Installation

# Cloner le projet
git clone https://github.com/bakwonken-hub/blog-api.git

# Aller dans le dossier
cd blog-api

# Installer les dépendances
npm install

# Configurer les variables d'environnement
cp .env.example .env

# Lancer l'application
npm start
